# Code Hero

Projeto criado utilizando a API da [Marvel](https://developer.marvel.com/) para exibir Personagens. Possuindo paginação dos resultados e funcionalidade de busca de personagens por nome.

## Tecnologias utilizadas

- [RecyclerView](https://developer.android.com/reference/androidx/recyclerview/widget/RecyclerView.html)
- [Vector Drawables](https://developer.android.com/guide/topics/graphics/vector-drawable-resources)
- [Firebase Remote Config](https://firebase.google.com/docs/remote-config)
- [Glide](https://github.com/bumptech/glide)
- [Retrofit](https://github.com/square/retrofit)
- [OkHttpClient](https://square.github.io/okhttp/)
